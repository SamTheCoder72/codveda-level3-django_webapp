# Django Task Manager (Codveda Level 3)

A full-stack web application built with Django and Bootstrap 5. This project was developed as the final Level 3 task for the Python Development Internship at Codveda Technologies.

## Features
* Secure User Authentication (Registration, Login, Logout)
* Session routing and access control
* Responsive UI built with Bootstrap 5
* SQLite backend database

## How to Run Locally
1. Clone the repository.
2. Install Django: `pip install django`
3. Run migrations: `python manage.py migrate`
4. Start the server: `python manage.py runserver`
